**MultiGen Pro | Ultra Edition ⚡**

MultiGen Pro — это мощный, безопасный и эстетичный инструмент «все в одном» для генерации цифровых данных. Приложение позволяет мгновенно создавать надежные пароли, уникальные игровые никнеймы, реалистичные имена и адреса электронной почты прямо в браузере.

!

🚀 Основные возможности

Password Gen: Создание криптостойких паролей с настройкой длины (до 64 символов), использованием спецсимволов, цифр и проверкой на читаемость.

Nickname Gen: Генерация ников в четырех стилях: Cyberpunk, Fantasy, Minimal, Brutal.

Identity Gen: Генерация реалистичных имен и фамилий (поддержка RU/US локализаций и выбора пола).

Gmail Gen: Быстрое создание шаблонов почтовых адресов (чистые или анонимные хэши).

Система безопасности: Визуальный индикатор энтропии (BITS) и уровня надежности пароля.

Лог генераций: История последних 30 действий сохраняется локально в вашем браузере.

Ultra UI: Современный стекляный интерфейс (Glassmorphism) с поддержкой темной и светлой тем.

🛠 Технологии

Frontend: Чистый HTML5 и современный JavaScript (ES6+).

Styling: Tailwind CSS для адаптивного и быстрого интерфейса.

Fonts: Plus Jakarta Sans & JetBrains Mono (для паролей).

Security: Использование window.crypto.getRandomValues для обеспечения истинной случайности (CSPRNG).

Storage: localStorage для сохранения истории и контактов после перезагрузки страницы.

📦 Установка и запуск

Приложение является Single File Application (SFA), что означает отсутствие необходимости в сложной установке:

Скачайте файл index.html.

Откройте его в любом современном браузере (Chrome, Firefox, Edge, Safari).

Для работы анимации фона и шрифтов требуется подключение к интернету (CDN).

💾 Сохранение данных

Приложение заботится о вашем удобстве:

История: Список сгенерированных данных не пропадает при обновлении страницы.

Контакты и настройки: Выбранные вами параметры и ссылки сохраняются в памяти браузера.

Приватность: Все данные генерируются на стороне клиента. Ни один пароль или имя не отправляется на сервер.
